## Java 풀이
### 다른 사람의 풀이
```java
class Solution {
  public int solution(int a, int b) {
    int answer = 0;

    while (b % 2 == 0) {
      b = b / 2;
    }

    for (b % 5 == 0) {
      b = b / 5;
    }

    if ((a / (double) b) % 1 == 0) {
      answer = 1;
    } else {
      answer = 2;
    }

    return answer;
  }
}
```

## JavaScript 풀이
### 다른 사람의 풀이
```javascript
function solution(a, b) {
  let n = 1;
  for (let i = 1; i <= Math.min(a, b); i++) {
    if (a % i === 0 && b % i === 0) n = i;
  }

  b /= n;
  while (b % 2 === 0) b /= 2;
  while (b % 5 === 0) b /= 5;

  return b === 1 ? 1 : 2;
}
```
